快速操作 / Quick Actions

  • 恢复内购 / Restore Purchase: 打开应用 → 自定义分类被锁时触发购买流程界面 → 选择“恢复” (如果有按钮)。 / If entitlement lost, trigger purchase screen and use "Restore" (if shown) or re‑purchase (Apple 不重复扣费 / you won't be charged twice).
  • 同步到小组件 / Widget Refresh: 创建或删除贴纸后等待 ~15 秒；必要时长按桌面进入编辑 → 移除并重新添加组件。 / After adding/deleting stickers wait ~15s; optionally re‑add the widget.
  • 删除分类 / Delete Custom Category: 分类管理界面长按或使用管理模式 → 选择删除 → 贴纸会移动到“未分类”。 / Long‑press or use manage mode → delete; stickers move to "Uncategorized".
  • 位置未记录 / Location Missing: iOS 设置 → 隐私与安全性 → 定位服务 → 找到 App → 设为 “使用 App 期间” 并开启“精确位置”。 / Settings → Privacy → Location → set to While Using & Precise.
  • 贴纸不见了 / Lost Stickers: 确认未卸载应用；检查是否更换设备且无转移；当前版本不做云同步。 / Ensure app not deleted; device migration not supported yet (no cloud sync).

常见问题 / FAQ

1. 内购已付费却再次提示购买 / Already purchased but asked to pay again

ZH: 这是 Apple 正常的重复购买界面；确认 Apple ID 后系统会显示“已购买”并恢复权限，不会二次扣费。请确保使用原来的 Apple ID。
EN: iOS may show the purchase sheet again; after confirming Apple ID it will say “You’ve already purchased this” and restore access without double‑charging.

2. 小组件不更新 / Widget not updating

ZH: 先打开主应用确保最近前台过；创建/删除贴纸后等待 10–30 秒。若仍不更新：重启设备或移除再添加组件。
EN: Open the main app (foreground refresh), wait 10–30s after changes. If still stale: reboot device or remove/re‑add the widget.

3. 为什么需要位置权限 / Why ask for location?

ZH: 仅在你拍摄贴纸并允许时，用一次性定位生成可读地址，帮助回忆创作场景，不上传服务器。可在设置中关闭。
EN: Used only (once) at capture—reverse geocoding for context. Not uploaded. You can disable in Settings anytime.

4. 如何彻底删除我的数据 / How to delete my data?

ZH: 删除贴纸即删除对应元数据；卸载应用即移除所有本地数据。
EN: Deleting a sticker removes its metadata; uninstalling the app removes all local data.

5. 为什么没有云同步 / No cloud sync?

ZH: 当前版本专注本地性能与隐私。后续若增加同步，会更新隐私政策。
EN: Focus on on‑device privacy first; if sync arrives later, policy will update.

6. 分类或贴纸操作延迟 / Slow category or sticker operations

ZH: 大量贴纸时首次分类加载需要额外时间；保持应用前台以完成缓存构建。
EN: Large libraries may need a first‑time metadata pass; keep the app foreground to finish caching.
